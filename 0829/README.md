## 🌟복잡도 분석
> 알고리즘이란, 유한한단계를 통하여 문제를 해결하는 방법
+ ⚡ 좋은 알고리즘이란,
  + 적은 메모리공간
  + 적은 시간
+ ⚡ 복잡도 점근적 표기
  + ⭐`Big-Oh` 최대⭐
  + `Big-Omega` 최소😕
  + `Big-Theta` 평균😕
+ ⚡ O(Big-Oh)-표기
  + 최대차수표기법 `O(2n^2+1) == O(n^2)`
  + 대략적 지표만 계산 가능
  + 상수횟수반복은 `O(1)`
  + 실행시간만의 효율을 나타내기 때문에 구현시간 + 난이도 + 등등의 다른 요소들도 고려해야한다.
+ ⚡ 자주사용하는 O-표기법
